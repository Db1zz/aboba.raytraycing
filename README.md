# MiniRT
This project is dedicated to ray tracing the simplest 3D geometric shapes using mathematics and optimisations magic.

## How-To-Use

1. Compile the project using `make bonus`
2. Run via `./miniRT scenes/amogus.rt` command
3. You can control camera using `w a s d` keys
